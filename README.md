# browser_label_manager
__基于 Vue2.x + TS 制作的自用网站导航项目__ 😄

__启动服务,进入 server 文件夹, yarn 安装依赖后, npm run dev__ 
## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
